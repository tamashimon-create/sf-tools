# ==============================================================================
# 削除対象リスト (remove-target.txt)
# ==============================================================================
#
# 【記述方法】
#   [files]   セクション: ファイルパスで指定するコンポーネント（通常はこちら）
#   [members] セクション: メンバー名で指定するコンポーネント（特殊ケース）
#
#   - 行頭が # の行はコメントとして無視されます
#   - 空行は無視されます
#   - [files] は force-app/main/default/ からの相対パスで記述します
#   - [members] は「メタデータ種別名:メンバー名」の形式で記述します
#
# 【重要な注意事項】
#   - 削除は元に戻せません。実行前に必ず内容を確認してください。
#   - データが紐づいているコンポーネントの削除は既存データの消失につながります。
#   - 標準オブジェクト・標準項目・標準プロファイルは削除できません。
#   - 削除後は Git リポジトリからもファイルを手動で削除してください。
#   - deploy-target.txt で指定可能でも、削除 API が非対応のため
#     このファイルに記述できないコンポーネントがあります（後述）。
#
# ==============================================================================
