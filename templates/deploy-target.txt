# ==============================================================================
# デプロイ対象リスト (deploy-target.txt)
# ==============================================================================
#
# 【記述方法】
#   [files]   セクション: ファイルパスで指定するコンポーネント（通常はこちら）
#   [members] セクション: メンバー名で指定するコンポーネント（特殊ケース）
#
#   - 行頭が # の行はコメントとして無視されます
#   - 空行は無視されます
#   - [files] は force-app/main/default/ からの相対パスで記述します
#     （VS Code で右クリック →「相対パスのコピー」が便利です）
#   - [members] は「メタデータ種別名:メンバー名」の形式で記述します
#     ※ 1ファイルに複数メンバーが集約されているコンポーネントを部分指定する場合に使用
#
# ==============================================================================
